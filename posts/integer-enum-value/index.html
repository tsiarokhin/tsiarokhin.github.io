Get integer enum value

July 19, 2022 One-minute read
Snippets
D365FO • SQL

Replace SalesStatus with your enum:

select t1.NAME, t2.ENUMID, t2.ENUMVALUE, t2.NAME EnumValueName from ENUMIDTABLE t1
	join ENUMVALUETABLE t2 on t1.ID = t2.ENUMID
	where t1.NAME = 'SalesStatus'
